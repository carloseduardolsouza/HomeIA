# Stack Tecnologica

Resumo da stack principal:

- Python (servicos de dados e ML)
- FastAPI (APIs)
- MQTT (mensageria IoT)
- InfluxDB, PostgreSQL e Redis (persistencia)
- Docker e Docker Compose (infra local)
- Grafana, Prometheus e Alertmanager (observabilidade)

Detalhamento tecnico em `ai-stack-architecture.md`.
